## Subtelne techniki przekierowania (per skill marek-core)

CTA mają być wpisane w narrację jako naturalny logiczny krok, nie sprzedażowy banner. Kluczowe wzorce:

1. **Open loop + walidowane narzędzie:** "Domowy pomiar PSQI to start — pełny obraz daje zestaw [BDI-II/STAI/ECR-R]" → link do psychozdrowie.online jako rozszerzenie pomiaru, nie produkt.
2. **Implementation intention:** "Jeśli przez dwa pomiary wynik PSQI > 10, to..." — frazujemy jako warunkowe instrukcje, czytelnik sam decyduje, kiedy klika.
3. **Empathic anchor przed CTA:** zawsze CTA poprzedzone empatycznym fragmentem rozumiejącym problem czytelnika, nigdy po liście danych.
4. **Autonomy preserved:** "to nie zastępuje X, daje przestrzeń, żeby..." — nigdy nie sugeruje, że produkt zastępuje terapię/lekarza.
5. **Specificity as authority:** wymieniamy konkretne testy (PSQI, BDI-II, NEO-FFI, ECR-R) — to klinicystyczny język, sygnał kompetencji bez przechwałek.

Każdy artykuł psychosen.pl ma maksymalnie 2 CTA: jedno inline (wpięte w treść), jedno na końcu (po esencji, jako naturalny "co dalej").
