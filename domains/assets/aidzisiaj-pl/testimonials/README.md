# Testimonials — aidzisiaj.pl

Wrzuć tutaj screeny z komunikatorów (Messenger, WhatsApp, Signal) od zadowolonych klientów/czytelników.

## Wymagane pliki (z domains/aidzisiaj-pl.yaml):

- `messenger-01.webp`
- `messenger-02.webp`
- `whatsapp-01.webp`
- `whatsapp-02.webp`
- `signal-01.webp`
- `signal-02.webp`

## Zasady:

1. Format: `.webp` preferowany (mniejszy), `.png`/`.jpg` OK (zmień rozszerzenie w YAML)
2. Rozmiar: max 800px szerokości (build.py nie skaluje testimoniali automatycznie)
3. ZANONIMIZUJ dane kontaktowe (numery, profile) PRZED uploadem
4. Upewnij się, że masz zgodę autora na publikację
5. Po dodaniu plików: `python build.py --domain aidzisiaj.pl --clean`
